Quickstart Guide

This guide will walk you through the essential steps to get up and running with our platform.

Step 1: Installation

Follow the instructions on the Installation page.

pip install our-platform

Step 2: Run Your First Model

Here's a simple example using Python:


from platform import Client

client = Client(api_key="YOUR_API_KEY")
response = client.generate("Tell me a joke.")
print(response)
